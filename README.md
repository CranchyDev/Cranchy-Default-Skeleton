# Cranchy-Default-Skeleton
My default skeleton/skeletal mesh that I use in most of my UE5 projects. Feel free to use and modify it at your own will! It's 100% rigged and has a few animations. Enjoy!

## What is within this repo?

1. The Blender file project (.blend).
2. An .fbx version of the file if you just want to quickly import it into Unreal Engine.
3. Skeleton is rigged, animated and prepared to setup IK (Inverse Kinematics) on the legs or feet!
